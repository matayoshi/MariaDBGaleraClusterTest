MariaDB Galera Clusterを試す
============================

MariaDB Galera Clusterを試すためのVagrantfile一式

## Description

MariaDB Galera Clusterを使ったシステム構成を試すためのVagrantfile一式です。  
このVagrantfileではWeb用VM1台、DB用VM2台の構成です。
